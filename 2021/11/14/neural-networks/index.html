Neural Networks

Neural networks are algorithms that attempt to mimic the function of the brain. The concept is raised in the 1980s, diminished in popularity in the 1990s, and is now implemented for many applications. This is a basic neuron model. It obviously contains three parts: the input wires, the neuron, and the output wires. The neuron processes the input wires into the output. Sometimes, the input wires would consist of an $x_0$. This term is usually 1 and is referred to as the bias term. The activation function refers to the non-linear function the neuron uses as the $g(z)$. Why do we emphasize the non-linear here? Well, if the functions are actually linear, the neural network would actually produce a linear model, which renders the neural network completely useless. The weights of a model are the same thing as the parameters of a model ($\theta$). In a neural network, the first layer is usually referred to as the input layer. The final layer is also referred to as the output layer. The term hidden layers are intermediate layers that sit in between the input layer and the output layer (not $x$ nor $y$). Note the hidden layer can contain multiple layers and they also contain bias units. $$\begin{align} a_i^{(j)} &= \text{“activation” of unit i in layer j} \newline \Theta^{j} &= \text{matrix of weights controlling function mapping from layer j to layer (j+1).} \end{align}$$ We can also determine the size of $\Theta$. \begin{align*}\text{If network has }s_j​ \text{units in layer j and }s_{j+1}\text{​ units in layer j+1, then }Θ^{(j)} \text{will be of dimension }s_{j+1}​×(s_j​+1).\end{align*} Our entire neural network can be described as: $$\begin{align*} a_1^{(2)} = g(\Theta_{10}^{(1)}x_0 + \Theta_{11}^{(1)}x_1 + \Theta_{12}^{(1)}x_2 + \Theta_{13}^{(1)}x_3) \newline a_2^{(2)} = g(\Theta_{20}^{(1)}x_0 + \Theta_{21}^{(1)}x_1 + \Theta_{22}^{(1)}x_2 + \Theta_{23}^{(1)}x_3) \newline a_3^{(2)} = g(\Theta_{30}^{(1)}x_0 + \Theta_{31}^{(1)}x_1 + \Theta_{32}^{(1)}x_2 + \Theta_{33}^{(1)}x_3) \newline h_\Theta(x) = a_1^{(3)} = g(\Theta_{10}^{(2)}a_0^{(2)} + \Theta_{11}^{(2)}a_1^{(2)} + \Theta_{12}^{(2)}a_2^{(2)} + \Theta_{13}^{(2)}a_3^{(2)}) \newline \end{align*}$$ [h2title]Forward Propagation[/h2title] Now let’s talk about forward propagation, but firstly let’s define a variable $z_k^{(j)}$ that includes all the terms in the $g$ function. $$\begin{align*}a_1^{(2)} = g(z_1^{(2)}) \newline a_2^{(2)} = g(z_2^{(2)}) \newline a_3^{(2)} = g(z_3^{(2)}) \newline \end{align*}$$

Where:

$$\begin{align}z_k^{(2)} = \Theta^{(1)}_{k,0}x_0 + \Theta^{(1)}_{k,1}x_1 +…+\Theta^{(1)}_{k,n}x_n \end{align}$$

Vectorization of $z$:

$$\begin{align*}x = \begin{bmatrix}x_0 \newline x_1 \newline\cdots \newline x_n\end{bmatrix} &z^{(j)} = \begin{bmatrix}z_1^{(j)} \newline z_2^{(j)} \newline\cdots \newline z_n^{(j)}\end{bmatrix}\end{align*}$$ $$\begin{align} z^{(j)} = \Theta^{(j-1)} a^{(j-1)} \end{align}$$

Now let’s substitute our $z$ back into our original function…

$$\begin{align} a^{(j)} = g(z^{(j)}) \end{align}$$

The computation for the next layer will be the following:

$$\begin{align} z^{(j+1)} = \Theta^{(j)} a^{(j)} \end{align}$$

Please note bias units can be added throughout the process.

Now we should have a perfect algorithm that allows us to calculate $a^{(x)}$:

$$\begin{align} z^{(x)} = \Theta^{(x-1)} a^{(x-1)} \end{align}$$ $$\begin{align} a^{(x)} = g(z^{(x)}) \end{align}$$

This algorithm, as it starts from the first layers, namely the input layers, and propagates all the way through the hidden layers to the output layers, is referred to as forward propagation. You may probably already understand what backward propagation is, but let’s slow it down a little, shall we? [h2title]Logic gates[/h2title] (Actually, this part is really easy. I’m gonna presume you already know everything about logic gates.) For a logic gate, let’s say we have binary inputs of $x_1$ and $x_2$ (0 and 1). Now now now, we can set our $\Theta$ in different values so when both $x_1$ and $x_2$ are 1, we output 1. To achieve this, bias terms are necessary to aid the hypothesis. $$\begin{align*}& h_\Theta(x) = g(-30 + 20x_1 + 20x_2) \newline \newline & x_1 = 0 \ \ and \ \ x_2 = 0 \ \ then \ \ g(-30) \approx 0 \newline & x_1 = 0 \ \ and \ \ x_2 = 1 \ \ then \ \ g(-10) \approx 0 \newline & x_1 = 1 \ \ and \ \ x_2 = 0 \ \ then \ \ g(-10) \approx 0 \newline & x_1 = 1 \ \ and \ \ x_2 = 1 \ \ then \ \ g(10) \approx 1\end{align*}$$

The only thing worth mentioning here is that $g(x) \approx 1$ when $x > 0$, and $\approx 1$ when $x < 0$. Now let’s construct the OR gate and the NOR gate JUST FOR FUN!!!!! $$\begin{align*}& h_\Theta(x) = g(10 - 20x_1 - 20x_2) \newline \newline & x_1 = 0 \ \ and \ \ x_2 = 0 \ \ then \ \ g(10) \approx 1 \newline & x_1 = 0 \ \ and \ \ x_2 = 1 \ \ then \ \ g(-10) \approx 0 \newline & x_1 = 1 \ \ and \ \ x_2 = 0 \ \ then \ \ g(-10) \approx 0 \newline & x_1 = 1 \ \ and \ \ x_2 = 1 \ \ then \ \ g(-30) \approx 0\end{align*}$$

$$\begin{align*}& h_\Theta(x) = g(-10 + 20x_1 + 20x_2) \newline \newline & x_1 = 0 \ \ and \ \ x_2 = 0 \ \ then \ \ g(-10) \approx 0 \newline & x_1 = 0 \ \ and \ \ x_2 = 1 \ \ then \ \ g(10) \approx 1 \newline & x_1 = 1 \ \ and \ \ x_2 = 0 \ \ then \ \ g(10) \approx 1 \newline & x_1 = 1 \ \ and \ \ x_2 = 1 \ \ then \ \ g(30) \approx 1\end{align*}$$ Please don’t contact me to help you find out which expression is which… Now let’s do something more advanced and try to create an XNOR gate. An XNOR gate can be constructed using two AND gates and one NOR gate. We now need to add a hidden layer to our neural network now. $$\begin{align*}\begin{bmatrix}x_0 \newline x_1 \newline x_2\end{bmatrix} \rightarrow\begin{bmatrix}a_1^{(2)} \newline a_2^{(2)} \end{bmatrix} \rightarrow\begin{bmatrix}a^{(3)}\end{bmatrix} \rightarrow h_\Theta(x)\end{align*}$$ According to the formula mentioned above in the passage, we can deduce the values for all the nodes. $$\begin{align*}& a^{(2)} = g(\Theta^{(1)} \cdot x) \newline& a^{(3)} = g(\Theta^{(2)} \cdot a^{(2)}) \newline& h_\Theta(x) = a^{(3)}\end{align*}$$ The $\Theta(1)$ will be the combination of an AND gate and a NOR gate. The $\Theta(2)$ would be an AND gate.
