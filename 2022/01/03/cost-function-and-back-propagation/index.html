Cost function and back propagation

My GPA is getting destroyed so I escape reality by writing neural network notes. G o o d   f o r   m e [h2title]Cost function for neural networks[/h2title]

Firstly let’s recap the regularized cost function of logistic regression, which we have revised multiple times.

$$J(\theta) = -\frac{1}{m} \sum_{i=1}^m\cdot \left[y^{(i)}\log(h_\theta(x^{(i)}))+(1-y^{(i)})\log(1-\theta(x^{(i)})) \right ]+ \frac\lambda {2m}\sum_{j=1}^{n} \theta^2_j$$

The neural network cost function is mostly similar, being slightly more complicated.

$$ J(\Theta)=-\frac{1}{m} \sum_{i=1}^{m} \sum_{k=1}^{K}\left[y_{k}^{(i)} \log \left(\left(h_{\Theta}\left(x^{(i)}\right)\right)_{k}\right)+\left(1-y_{k}^{(i)}\right) \log \left(1-\left(h_{\Theta}\left(x^{(i)}\right)\right)_{k}\right)\right]+\frac{\lambda}{2 m} \sum_{l=1}^{L-1} \sum_{i=1}^{s_l} \sum_{j=1}^{s_{l+1}}\left(\Theta_{j, i}^{(l)}\right)^{2} $$ I know what are you guys thinking. This is only slightly complicated? Actually, if you understand the model of a neural network, you should be able to understand this. I just realized I forgot to introduce the new variables. My bad.

k: stands for the total number of output units in the neural network.

$s_l$: stands for the number of nodes in a layer

l: stands for the number of layers

As now we’re dealing with the entire neural network here, we need to iterate through every output unit to calculate the cost of the entire network. Thus in the first part of the cost function, we introduce the double sum $\sum_{k=1}^{K}$ for this particular purpose. For the regularization component, here we utilize a triple sum(what). As we talked about before in the neural network model, the size of the $\theta$ matrix is equal to the number of nodes in the next layer * the number of nodes in this layer. We iterate the matrix two-dimensionally and square every term and add it to our cost. We need to do this for every layer so this is what the $\sum_{l=1}^{L-1}$ is for. [h2title]Back propagation[/h2title] We combine forward propagation and backpropagation to minimize the cost of our neural network. Firstly, we complete forward propagation using the train sets (or the training examples) and derive the output nodes. This process will derive the values for all the activation units in the neural network. The next thing we do is to derive $\delta$ . This is a new concept, as it represents the error values in this neural network. You might suggest that this is already represented by the cost function, but actually, the error values are similar to the derivatives of the cost functions. We’ll introduce this more later on. Anyways, the formula to calculate $\delta$ when it’s at the output layer is $\delta^{(L)}=a^{(L)}-y^{(t)}$. This is pretty easy to understand as this just calculates the difference between the activation values at the output layer and the actual value in the training examples. Notice how this is a vectorized representation so the $\delta^{(L)}$ is a vector. Now here comes the interesting part. We use the $\delta^{(L)}$ at the output layer to calculate the $\delta^{(L-1)}$ and its former terms by the formula below. $$ \delta^{(l)}=\left(\left(\Theta^{(l)}\right)^{T} \delta^{(l+1)}\right) \cdot * g’(z{(l)}) $$ To be honest this is very perplexing. What we’re actually doing here is to multiply the next layer’s error values with the theta of this layer’s. We then do an element-wise multiplication with the derivatives of the activation function (sigmoid function here is) with the input values of this layer. The derivatives here can be represented also through $a^{(l)} \cdot *(1-a^{(l)})$, so we can rewrite the formula into… $$ \delta^{(l)}=\left(\left(\Theta^{(l)}\right)^{T} \delta^{(l+1)}\right) \cdot * a^{(l)} \cdot *\left(1-a^{(l)}\right) $$ Through this function, we can derive all the error values of the neural network up to $\delta^{(2)}$. We don’t need to compute $\delta^{(1)}$ because the error values are not needed in the input layer. After we compute the error values for one training example, we then use an accumulator matrix to add up the error values. We call this matrix $\Delta$. The vectorized approach of the accumulation function is $\Delta^{(l)}:=\Delta^{(l)}+\delta^{(l+1)}\left(a^{(l)}\right)^{T}$. We then repeat the process for every training example, for a total of m times. After this, we can finally deduce the partial derivatives of the cost function by the last formula below.

$\begin{aligned} D_{i, j}^{(l)} &:=\frac{1}{m}\left(\Delta_{i, j}^{(l)}+\lambda \Theta_{i, j}^{(l)}\right), \text { If } \mathrm{j} \neq 0 . \\ D_{i, j}^{(l)} &:=\frac{1}{m} \Delta_{i, j}^{(l)} \text { If } \mathrm{j}=0 \end{aligned}$

The last process here is to divide the $\Theta$ matrix by m so it computes the average for all the error values. The regularization term here may also seem very familiar to you, and as for always, we don’t consider the part when j = 0. To be honest, the backpropagation algorithm is insanely challenging to understand. Try your best at following the steps of backpropagation and maybe you can gain a deeper understanding of the concept here.
